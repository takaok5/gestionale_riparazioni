# PRD - Gestionale Riparazioni

## Vision
Sistema gestionale per riparazioni con autenticazione a ruoli (ADMIN, TECNICO, COMMERCIALE).

## Functional Requirements

- FR-001: Login utente con username/password, gestione errori credenziali, account disabilitato e rate-limit.
- FR-002: Rinnovo sessione tramite refresh token con validazione token e controllo account attivo.

## Non-Functional Requirements

- NFR-001: API backend tipizzate TypeScript con test automatizzati.
- NFR-002: Sicurezza autenticazione con JWT e gestione errori coerente.

## Epic List

- Epic 1: Autenticazione e Gestione Sessione
