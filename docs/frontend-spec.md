# Frontend Spec - Gestionale Riparazioni

## Scope attuale

- Pagina login con invio credenziali a /api/auth/login.
- Gestione token di sessione lato client.

## Requisiti UI

- Messaggi errore chiari per credenziali non valide e account disabilitato.
- Redirect a schermata autenticata dopo login valido.
- Supporto rinnovo sessione tramite endpoint /api/auth/refresh.
